Privacidad

Tus respuestas no salen del navegador.

Pausa es un sitio informativo estático. Sus herramientas se ejecutan en tu dispositivo y no transmiten lo que escribes.

Qué recoge este sitio

Cuentas
No hay autenticación ni registro.
Base de datos
No existe una base de datos.
Cookies
El proyecto no establece cookies.
Analítica
No utilizamos analítica de comportamiento ni píxeles.
Formularios
Ningún formulario envía datos a un servidor.
APIs de IA
Los ejemplos son texto para copiar. Este sitio no llama a una IA.
Tipografías
Usamos fuentes del sistema; no se solicitan a un proveedor externo.

Cómo funcionan las herramientas

El check-in, los organizadores y el constructor de resumen funcionan con JavaScript dentro de esta pestaña. Al recargar la página, la información introducida desaparece.

La función de copiar usa el portapapeles del navegador. El texto va directamente al portapapeles de tu dispositivo.

Qué no hace este sitio

  • No ofrece diagnóstico ni tratamiento.
  • No sustituye una consulta profesional.
  • No garantiza resultados clínicos.
  • No recopila datos identificables.
  • No comparte respuestas con terceros.

Transparencia

Si algo cambia, esta página también

Antes de incorporar almacenamiento, servicios externos o analítica, actualizaremos esta información y la fecha de revisión.
